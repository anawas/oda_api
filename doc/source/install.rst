Installation
============

Download code
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
https://github.com/andreatramacere/oda_api/archive/stable.tar.gz

Installation
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1) Anaconda
    * `while read requirement; do conda install --yes $requirement; done < requirements.txt`
    * `python setup.py install`

2) PIP
    * `pip install -r requirements.txt`
    * `python setup.py install`
